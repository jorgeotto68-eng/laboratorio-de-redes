Un guía para la subred

¿Qué es CIDR?

CIDR (Classless Inter-Domain Routing) es un método para asignar direcciones IP de forma más eficiente. Permite crear subredes de tamaño variable, a diferencia de las antiguas clases de IP (A, B y C).

Clases de Direcciones IP

Clase A: Prefijo de 8 bits (ej. /8)

Clase B: Prefijo de 16 bits (ej. /16)

Clase C: Prefijo de 24 bits (ej. /24)

Cálculo de Bits y Subredes

El valor del prefijo CIDR (por ejemplo, /24) indica cuántos bits de la dirección IP son parte de la porción de red.

Para calcular el número de subredes, se usa la fórmula: $2^n$ (donde 'n' es el número de bits prestados).

Conclusión

La tabla CIDR es una herramienta fundamental en la administración de redes. Te permite calcular rápidamente la máscara de subred, el número de subredes y la cantidad de hosts disponibles, lo que facilita la planificación y el despliegue de redes eficientes.
