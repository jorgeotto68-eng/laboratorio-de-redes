El Modelo OSI

Un marco para la comunicación en red

¿Qué es el Modelo OSI?

El Modelo OSI (Open Systems Interconnection) es un marco conceptual de siete capas que estandariza la forma en que los sistemas de red se comunican. Cada capa tiene una función específica para la transmisión de datos.

Las 7 Capas del Modelo OSI

  • Capa 7: Aplicación (Datos)
  • Capa 6: Presentación (Datos)
  • Capa 5: Sesión (Datos)
  • Capa 4: Transporte (Segmentos)
  • Capa 3: Red (Paquetes)
  • Capa 2: Enlace de Datos (Tramas)
  • Capa 1: Física (Bits)

Capa 7: Aplicación

Interactúa directamente con el software del usuario. Proporciona servicios de red a las aplicaciones.

PDU: Datos

Protocolos: HTTP, HTTPS, FTP, SMTP, DNS

Capa 6: Presentación

Se encarga de la sintaxis y la semántica de los datos, incluyendo la codificación, cifrado y compresión de la información.

PDU: Datos

Protocolos: JPEG, GIF, TLS, SSL

Capa 5: Sesión

Establece, gestiona y finaliza las conexiones (sesiones) entre las aplicaciones. Sincroniza el diálogo entre las capas de presentación.

PDU: Datos

Protocolos: L2TP, PPTP, RPC

Capa 4: Transporte

Divide los datos en segmentos y los reensambla en el destino. Asegura la entrega de los datos de forma confiable.

PDU: Segmentos (TCP), Datagramas (PDU)

Protocolos: TCP, PDU

Capa 3: Red

Se encarga del direccionamiento lógico (IP) y el enrutamiento de los paquetes a través de diferentes redes.

PDU: Paquetes

Protocolos: IP, ICMP, OSPF, EIGRP

Capa 2: Enlace de Datos

Proporciona la transferencia de datos entre nodos adyacentes. Utiliza direcciones físicas (MAC).

PDU: Tramas

Protocolos: Ethernet, PPP

Capa 1: Física

Define las especificaciones mecánicas y eléctricas para la transmisión de datos binarios (bits) a través de un medio físico.

PDU: Bits

Protocolos: USB, Bluetooth, 802.11

Encapsulación: El Viaje de los Datos

El proceso de encapsulación es cómo los datos se empaquetan en cada capa del modelo OSI a medida que viajan desde el origen hasta el destino.

  • **Datos** (Capas 7, 6, 5)
  • **Segmento** (Capa 4)
  • **Paquete** (Capa 3)
  • **Trama** (Capa 2)
  • **Bits** (Capa 1)

Modelo OSI vs. Modelo TCP/IP

  • **Modelo OSI (7 capas):** Aplicación, Presentación, Sesión, Transporte, Red, Enlace de Datos, Física.
  • **Modelo TCP/IP (4 capas):** Aplicación, Transporte, Internet, Acceso a la Red.

El modelo TCP/IP es más práctico y el que se usa en Internet, combinando algunas de las capas de OSI.
