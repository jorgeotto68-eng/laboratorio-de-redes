Telecomunicaciones y Redes de Datos

Redes: Conjunto de equipos de comunicaciones y medio de transmisión que posibilitan el envío de información. Un número de computadoras autónomas separadas pero interconectadas.

Telecomunicaciones: Toda transmisión, emisión o recepción de signos, señales, escritos, imágenes, sonidos o informaciones de cualquier naturaleza por hilo, radioelectricidad, medios ópticos u otros sistemas electromagnéticos.

¿Por qué necesitamos una red?

  • Para intercambiar información
  • Para utilizar aplicaciones centralizadas
  • Para tener acceso a distintos servicios
  • Para compartir recursos
